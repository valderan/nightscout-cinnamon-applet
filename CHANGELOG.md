# Changelog

All notable changes to this project will be documented in this file.

## [0.3.0] - 2026-02-03

### Added
- High glucose threshold setting with customizable color
- Low glucose threshold setting with customizable color
- Color indication on panel when glucose reaches thresholds
- Russian translations for new settings

### Changed
- Settings UI now includes "Glucose thresholds" section

## [0.2.0] - 2024-04-09

### Changed
- Simplified for CGM-only setups (removed pump support)
- Updated for Nightscout API v14+/v15+ compatibility

## [0.1.0] - Initial release

### Added
- Basic Nightscout blood glucose display
- Trend arrows support
- Missing readings alerts
- Multi-language support
